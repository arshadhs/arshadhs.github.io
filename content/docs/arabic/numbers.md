## 11–19 (أَحَدَ عَشَرَ إِلَى تِسْعَةَ عَشَرَ)

Numbers **11–19 are special**:
- They are said as **“unit + ten”**
- No **وَ (and)** is used

| Number | Arabic Numeral | Arabic |
|-------:|:--------------:|--------|
| 11 | ١١ | أَحَدَ عَشَرَ |
| 12 | ١٢ | اِثْنَا عَشَرَ |
| 13 | ١٣ | ثَلَاثَةَ عَشَرَ |
| 14 | ١٤ | أَرْبَعَةَ عَشَرَ |
| 15 | ١٥ | خَمْسَةَ عَشَرَ |

## 21–29 (الوَاحِد وَالعِشْرُون → التِّسْعَة وَالعِشْرُون)

From **21 onwards**, Arabic uses:
> **unit + وَ + ten**

| Number | Arabic Numeral | Arabic |
|-------:|:--------------:|--------|
| 21 | ٢١ | وَاحِد وَعِشْرُون |
| 22 | ٢٢ | اِثْنَان وَعِشْرُون |
| 25 | ٢٥ | خَمْسَة وَعِشْرُون |
| 29 | ٢٩ | تِسْعَة وَعِشْرُون |

## Numbers Between the Tens (31–99)

Same rule applies:
> **unit + وَ + ten**

Examples:
- 34 → **٣٤** → أَرْبَعَة وَثَلَاثُون  
- 58 → **٥٨** → ثَمَانِيَة وَخَمْسُون  
- 76 → **٧٦** → سِتَّة وَسَبْعُون

## Pattern Summary

- **1–10** → basic forms  
- **11–19** → unit + ten (no وَ)  
- **21–99** → unit + وَ + ten  
- **100+** → hundreds + وَ + rest  
- **1000+** → thousands + rest
