# Useful Gradient Identities

{{% colour RED %}}
\[
\nabla (a^T x) = a
\]
{{% /colour %}}

{{% colour RED %}}
\[
\nabla (x^T A x) = (A + A^T)x
\]
{{% /colour %}}

If A symmetric:

{{% colour RED %}}
\[
\nabla (x^T A x) = 2Ax
\]
{{% /colour %}}

These are heavily used in **optimisation**.
